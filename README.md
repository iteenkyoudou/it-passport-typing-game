# ITパスポート試験対策タイピングゲーム 🎮

ITパスポート試験の学習を楽しくゲーム感覚で進められるタイピングゲームです。

## 🎯 特徴

- **ゲーミフィケーション**: レベルアップ、カード収集、称号システム
- **段階的学習**: Stage1〜12まで順番に学習
- **ボスバトル**: 最終ステージでボスに挑戦
- **詳細な解説**: 各問題に丁寧な解説付き

## 🚀 起動方法

### 推奨: Python3で起動（Node.js不要）
```bash
# プロジェクトディレクトリに移動
cd 37_ITパスポートタイピングゲーム

# HTTPサーバーを起動（Python3のみ必要）
python3 -m http.server 8000

# ブラウザで以下のURLにアクセス
http://localhost:8000/course.html
```

**💡 ポート8000が使用中の場合:**
```bash
# 使用中のプロセスを確認
lsof -i :8000

# プロセスを終了（PIDは上記コマンドで確認）
kill <PID>
```

### 別の方法: npmスクリプトで起動（Node.js/npmが必要）
```bash
# Node.js/npmがインストールされている場合のみ
npm start
```
※ `npm start` は内部で `python3 -m http.server 8000` を実行しています

## ⚠️ 注意事項

- **直接HTMLファイルを開かないでください**
  - `file://`プロトコルではJSONファイルが読み込めません
  - 必ず上記の方法でHTTPサーバー経由でアクセスしてください

## 📁 ファイル構成

```
.
├── course.html          # メインのゲーム画面
├── course/              # ゲームデータ（JSON形式）
│   ├── chapter1_stages1-4.json
│   ├── chapter1_stages5-8.json
│   ├── chapter1_stages9-11.json
│   └── chapter1_stage12_boss.json
└── 00_MEMO/            # 開発メモ
```

## 🎮 ゲームの進め方

1. Stage 1から順番にクリア
2. 各ステージで問題に答えてタイピング練習
3. 経験値を貯めてレベルアップ
4. カードを集めてコレクションを完成
5. Stage 11をクリアしたらボスバトルに挑戦！

## 🛠️ 開発者向け

データの編集は`course/`ディレクトリ内のJSONファイルで行ってください。
HTMLファイル内のハードコードされたデータは削除されています。
