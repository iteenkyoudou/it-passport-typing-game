# ITパスポート学習ゲーム

ITパスポート試験の学習を楽しくゲーム感覚で進められるWebアプリです。

---

## 2つのプロジェクトについて

このリポジトリには**2つの独立したプロジェクト**が共存しています。

| プロジェクト | ファイル | 概要 |
|-------------|---------|------|
| **タイピングゲーム** | `index.html` | 穴埋めタイピング形式でIT用語を覚える |
| **対策講座** | `course.html` | ステージ制で過去問を学習する |

### タイピングゲーム（index.html）
- 穴埋め問題をタイピングで回答
- ドラクエ風モンスターバトル演出
- 練習モード / 本気モード（HP制）
- ボスバトル・称号・ランキング機能

### 対策講座（course.html）
- Stage 1〜12の段階的学習
- 問題 → 解説 → 練習問題 → カード獲得
- レベルアップ・称号システム
- 章ボスバトル

> 詳細な仕様は `99_仕様/` フォルダを参照してください。

---

## 起動方法

```bash
# プロジェクトディレクトリに移動
cd 37_ITパスポートタイピングゲーム

# HTTPサーバーを起動
python3 -m http.server 8000
```

### アクセスURL
| プロジェクト | URL |
|-------------|-----|
| タイピングゲーム | http://localhost:8000/index.html |
| 対策講座 | http://localhost:8000/course.html |

### ポート8000が使用中の場合
```bash
lsof -i :8000      # 使用中のプロセスを確認
kill <PID>         # プロセスを終了
```

## 注意事項

### 直接HTMLファイルを開かないでください
- `file://`プロトコルではJSONファイルが読み込めません
- 必ずHTTPサーバー経由でアクセスしてください

### プロジェクト共存について
- 2つのプロジェクトは**独立して動作**します
- 画像・音声ファイルは**共通リソース**として共有しています
- 改修時は対象プロジェクトを明確にしてください
