# 修正依頼一覧

## 1. レベル上限の変更（course.html）

### 現状
- 仕様上は最大Lv999だが、2章の途中でLv70に到達してしまう
- 9章まであるのにレベルが早く上がりすぎる

### 修正内容
- レベル上限を **Lv999** に確定（コード上は既にMAX_LEVEL=999）
- EXP曲線のバランス調整が必要か要検討（現状の`expForLevel`式だと上がりが早すぎる可能性）

### データ影響
- なし（既存セーブデータの構造は変わらない）

---

## 2. ボス問題の累積出題（course.html）

### 現状
- 各章ボスはその章の `practiceQuestions` のみから出題

### 修正内容
- ボス問題を**累積式**にする

| 章ボス | 出題範囲 |
|--------|---------|
| 1章ボス | 1章のみ |
| 2章ボス | 1章 + 2章 |
| 3章ボス | 1章 + 2章 + 3章 |
| ... | ... |
| 9章ボス | 全章（1〜9章の総復習） |

### データ影響
- なし（問題の選択ロジックのみ変更。COURSE_DATAやBOSS_DATAの構造は変えない）

---

## 3. ボス戦の必要回答数を段階的に増加（course.html）

### 現状
- 出題数はランダム **3〜6問**（全章共通）
- クリア条件は **全問正解**

### 修正内容
- 章が進むごとに出題数を段階的に増やす
- 具体的な数値は要相談（例: 1章3〜4問 → 9章12〜15問）
- クリア条件（全問正解）は変えない

### データ影響
- なし（ゲームロジックのみの変更）

---
